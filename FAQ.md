# Frequently Asked Questions

## Should I learn Java before Spring Boot?

Yes.

A solid understanding of Java is essential before learning Spring Boot.

---

## Do I need SQL?

Yes.

Every backend developer should understand relational databases and SQL.

---

## Is Spring Boot enough to get a job?

No.

You should also know:

- Java
- SQL
- Git
- REST APIs
- JPA
- Spring Security
- Docker

---

## Should I learn Docker?

Yes.

Most backend applications are deployed using containers.

---

## Should I learn Microservices first?

No.

Master monolithic Spring Boot applications before moving to distributed systems.

---

## Which Java version should I use?

Java 21 (LTS) is recommended.
